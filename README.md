# My Mate

Web localStorage를 이용한 일정관리 웹 애플리케이션입니다. [Link](http://kyw624.github.io/my-mate)

<br />

## **주요 기술**

- JavaScript
- HTML
- CSS
- SCSS

<br />

## **제작 기간**

2020-08-16 ~ 2020-09-01

<br />

## **기능 소개**

1. 반응형 지원
2. 시계
3. 리스트
   - 기본 리스트 (제거 불가 / 리셋 가능)
     - Important: 중요 일정만을 보여주는 리스트
     - Tasks: 기본으로 제공되는 리스트
   - 리스트 추가
   - 리스트 제거
   - 리스트 초기화
4. 할 일 목록
   - 중요 일정 스위치
   - 아이템 삭제
